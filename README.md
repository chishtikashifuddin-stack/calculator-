# 🧮 Simple Calculator (Python Project)

A simple command-line calculator built using Python.

This calculator performs basic mathematical operations such as:

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division

The calculator also supports continuous calculations and reset functionality using `cls`.

---

# 🔄 Reset Calculator

To clear all previous calculations and start a new calculation:

```txt
cls
```

Example:

```txt
ENTRE NUMBER:- cls
START NEW :
```
